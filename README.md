## Pasos para utilizarlo

**1.** Clonar

**2.** Acceder a la primera carpeta "example" y copiar su path

**3.** Abrir cmd con la direccion anterior

**4.** Crear virtual environment *py -m venv env*

**5.** Activar virtual environment *.\env\Scripts\activate*

**6.** Instalar todos los requerimientos *pip install -r requirements*

**7.** Levantar servidor *python manage.py runserver*

**8.** Abrir postman y ejecutar los diferentes post/get demostrados en el video
